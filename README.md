<h1>Azure Sentinel Tutorial with Live Mapping (SIEM)</h1>
This tutorial outlines the use and implementation of Azure Sentinel, as well as mapping live cyber attacks.<br />


<h2>Environments and Technologies Used</h2>

- Microsoft Azure (Virtual Machines/Compute)
- Remote Desktop
- Active Directory Domain Services
- PowerShell
- www.ipgeolocation.io API Key.

<h2>Operating Systems Used </h2>

- Windows 10 (21H2)

<h2>High-Level Deployment and Configuration Steps</h2>

- Setting up our resources in Azure.
- Using PowerShell to extract IP info from logs.
- Sending IP info to geolocation API.
- Visualizing the attacks on a world map with live updates.
